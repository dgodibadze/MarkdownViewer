# Screenshots

The current shots were captured from the Windows build. To refresh them (or replace with
macOS captures), drop PNGs here with these exact names — they appear in the main `README.md`:

| Filename | What it should show |
|---|---|
| `split.png` | Split mode — raw markdown on the left, rendered preview on the right |
| `about.png` | The About window (icon, name, Changelog / Architecture / Design buttons) |

Tip: on macOS, `⇧⌘4` then press **Space** and click a window to capture just that window
with a clean drop shadow (transparent background) — ideal for these.
